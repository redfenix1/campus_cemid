Configuración Wi-Fi
CAMPUS_CEMID

Sigue los siguientes pasos para poderte conectar en la red Wi-Fi CAMPUS_CEMID

Sistema Operativo

Esta es una guía que puede variar dependiendo del fabricante del dispositivo y la versión del sistema operativo
  • Android
  • iPhone
  • Windows
  • Linux

Paso 1

Activa el Wi-Fi en tu dispositivo y selecciona la red CAMPUS_CEMID.

Ejemplo dispositivo Samsung

Paso 2

Selecciona alguna de las opciones para obtener mayor información

1

Ingresa tu correo de Microsoft 365.

2

Digita tu Número de documento.

3

Selecciona la opción Ver más u Opciones avanzadas.

Ejemplo dispositivo Samsung

Paso 3

Selecciona alguna de las opciones para obtener mayor información

1

En Certificado CA selecciona la opción Usar certificados del sistema

2

En el campo Dominio escribe radius.cemid.org

Ejemplo dispositivo Samsung

Paso 4

Selecciona alguna de las opciones para obtener mayor información

1

En Método EAP selecciona la opción TTLS

2

Para la opción Estado de certificado en línea elige Estado de la solicitud o Solicitar estado del certificado.

3

Para finalizar, en Autenticación de fase 2 selecciona PAP.

Ejemplo dispositivo Samsung

Paso 5

¡Felicidades! estás conectado a CAMPUS_CEMID.

Ejemplo dispositivo Samsung
